## 第三天  
* 问题数组: 8 3 22 15 ( ) 35 50 ( ) 64 99 … 
* 这其实是一个等差数组。我们可以将这个数组分为两个部分。

|第一部分|8|22|( )|50|64|
|-------- | :--------: | :--------: |-------- | :--------: | :--------: |
|规律:相差14|8|8+14=22|22+14=(36)|36+14=50|50+14=64|

|第二部分|3|15|35|( )|99|
|-------- | :--------: | :--------: |-------- | :--------: | :--------: |
|规律:相差数每次加8|3|3+4+8=15|15+12+8=35|35+20+8=(63)|(63)+28+8=99|
